# Emmanuel Kgaphola – Portfolio Website

A modern, responsive personal portfolio website built to showcase my skills, projects, and professional experience in Cloud Administration, IT Support, and Software Development.

Live Site: https://kgaphola.vercel.app

---

## Overview

This portfolio presents my background as an IT Support and Cloud Operations Specialist. It includes:

- A summary of my technical skills in cloud computing, automation, and software development
- A projects section highlighting hands-on experience in web development and IT systems
- An interactive skills dashboard covering Cloud, Networking, Automation, and Software Development
- A contact section for professional inquiries and collaboration

---

## Built With

- HTML5 – Semantic and accessible markup
- CSS3 / SCSS – Modern, responsive styling
- JavaScript (ES6) – Client-side interactivity and animations
- Font Awesome – Icons used across the interface
- Vercel – Hosting and continuous deployment

---

## Features

- Responsive and professional user interface
- Interactive sections with animations and expandable content
- Strong focus on cloud administration and IT operations
- Highlights of key achievements and system metrics
- Simple and accessible contact section
- Easy to customize and extend
